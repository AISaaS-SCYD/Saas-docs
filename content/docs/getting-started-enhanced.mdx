# Getting Started

Get up and running with our API in just a few minutes.

<Callout type="info">
**Prerequisites**: Make sure you have an active account and have generated your API credentials from the dashboard.
</Callout>

## Quick Start Guide

<Steps>
<Step>
### Create Your Account

Sign up at [dashboard.example.com](https://dashboard.example.com) and verify your email address.

</Step>

<Step>
### Generate API Credentials

Navigate to Settings â†’ API Keys and create a new API key. Keep this secure - you won't be able to see it again!

<Callout type="warn">
Store your API key in environment variables or a secure secrets manager. Never commit it to version control.
</Callout>

</Step>

<Step>
### Make Your First Request

Try making a simple API call to verify your setup:

<Tabs items={['Node.js', 'Python', 'cURL']}>
  <Tab value="Node.js">
```javascript title="index.js"
const fetch = require('node-fetch');

async function getUsers() {
  const response = await fetch('https://api.example.com/v1/users', {
    headers: {
      'Authorization': `Bearer ${process.env.API_KEY}`,
      'Content-Type': 'application/json'
    }
  });

  const data = await response.json();
  console.log(data);
}

getUsers();
```
  </Tab>

  <Tab value="Python">
```python title="main.py"
import os
import requests

def get_users():
    response = requests.get(
        'https://api.example.com/v1/users',
        headers={
            'Authorization': f'Bearer {os.getenv("API_KEY")}',
            'Content-Type': 'application/json'
        }
    )

    data = response.json()
    print(data)

get_users()
```
  </Tab>

  <Tab value="cURL">
```bash title="request.sh"
curl -X GET https://api.example.com/v1/users \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```
  </Tab>
</Tabs>

</Step>

<Step>


</Step>
</Steps>

## Common Issues

<Callout type="error">
**401 Unauthorized**: Check that your API key is correct and properly formatted in the Authorization header.
</Callout>

<Callout type="warn">
**429 Rate Limited**: You've exceeded the rate limit. Implement exponential backoff and respect the `Retry-After` header.
</Callout>
