## Getting Started

Welcome! This guide will help you set up your account and make your first API call in just a few minutes.

### Prerequisites

Before you begin, make sure you have:

- A valid account on our platform
- Node.js 18+ or Python 3.8+ installed
- Basic understanding of REST APIs

### Step 1: Create an Account

1. Visit our [signup page](https://example.com/signup)
2. Fill in your details and verify your email
3. Complete the onboarding process

### Step 2: Get Your API Key

Once logged in:

1. Navigate to **Settings** > **API Keys**
2. Click **Generate New Key**
3. Copy your API key and store it securely

```bash
export API_KEY="your_api_key_here"
```

### Step 3: Make Your First API Call

Here's a simple example to test your setup:

```javascript
// Node.js example
const response = await fetch('https://api.example.com/v1/user', {
  headers: {
    'Authorization': `Bearer ${process.env.API_KEY}`,
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python
# Python example
import requests
import os

headers = {
    'Authorization': f'Bearer {os.getenv("API_KEY")}',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.example.com/v1/user', headers=headers)
print(response.json())
```

## Troubleshooting

### Common Issues

**API Key Not Working**
- Ensure the key is copied correctly without extra spaces
- Check that the key hasn't been revoked
- Verify you're using the correct environment (production/staging)

**Rate Limiting**
- Free tier: 100 requests/minute
- Pro tier: 1000 requests/minute
- Enterprise: Custom limits
